css使用笔记

在项目中发现不少效果是css3实现（用js实现确实会比较麻烦）

  • 1.css3动画相对于js下写法上比较简单，
  • 2.性能上，浏览器对css3的动画做过响应的优化（关于硬件加速的好坏 →）

animation的使用

大小和颜色循环变化 
	  -webkit-animation:fadeOut 1s infinite ease-in-out;
		  animation:fadeOut 1s infinite ease-in-out;
		/*建议不要使用form，to，使用百分比控制 */
		@keyframes fadeOut{
		    0% {transform: scale(0.1);background-color: yellow;}
		    25% {transform: scale(0.25);background-color: orange;}
			50% {transform: scale(0.5);background-color: red; }
			75% {transform: scale(0.75);background-color: #1F705A;}
			100% {transform: scale(1.0);background-color: blue; }
		}
