梳理一下几个尺寸单位的理解

  • px 像素点 显示屏(不考虑os对高分辨率屏幕的优化，如 Retina屏的显示处理)

  • em 基于父元素的尺寸，em继承父元素的大小 example 

  1. body {
  2.     font－size:62.5%;
  3.     /*font－size:10px;上下两句的效果是一样*/
  4.     /*body里面的元素 原来是 12px 写成1.2em即可*/
  5. }
  • rem css3属性（IE8 IE8- 不支持） 基于根元素（html元素） example 
  1.     html {
  2.             font-size: 20px;/*调节根元素大小可以达到放大缩小的效果*/
  3.         }
  • dpi Dots Per Inch（每英寸所打印的点数）
  • ppi 显示屏的像素密度
    计算公式


  • dppx(每像素包含点的数量（dots per pixel）) 貌似是css3 用来判断Retina屏的
  1. <!DOCTYPE html>
  2. <html lang="zh-cmn-Hans">
  3. <head>
  4. <meta charset="utf-8" />
  5. <style>
  6. @media screen and (min-resolution: 2dppx) {
  7.     .normal{display:none;}
  8. }
  9. @media screen and (min-resolution: 1dppx) and (max-resolution: 1.9dppx) {
  10.     .retina{display:none;}
  11. }
  12. </style>
  13. </head>
  14. <body>
  15. <p class="retina">视网膜屏</p>
  16. <p class="normal">普通屏</p>
  17. </body>
  18. </html>
